# PhychoTest

## Venv Setup
### Mac
To install Virtualenv:
```bash
$ brew install python3
```
```bash
$ pip3 install virtualenv
```
To create venv:
```bash
$ virtualenv -p python3 venv
```
To activate venv:
```bash
$ source venv/bin/activate
```
To deactivate venv:
```bash
$ deactivate
```

## Commands
To run server locally:
```bash
$ python manage.py runserver
```
